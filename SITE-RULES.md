# AI Fellows Site — Copy & Style Rules

## Voice / Tone
- No em-dashes (—) anywhere — use commas, periods, or rewrite
- No geographic callouts like "colleges that get skipped" — too on-the-nose, sounds condescending
- No founder name (Angshuman Rudra) anywhere on the site — the program is the identity, not the person
- Keep language direct and practitioner-facing — no nonprofit clichés

## Specific things to avoid
- "for the colleges that get skipped" — remove
- "AI hiring pipelines cluster around the same 20 schools" — too aggressive, reframe
- Any first/last name of the mentor — use title + org only if needed, or omit entirely
